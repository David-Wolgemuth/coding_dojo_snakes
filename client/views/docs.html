Documentation

Function must return a cardinal direction: "n", "e", "s", "w". ("?" can be returned to go straight) Available Variables:
  • snake
    • letter
      • type: string
      • character that represent snake on map
      • ex: "b"
    • saved
      • type: object
      • values stored in "saved" key can be accessed later (anything stored in snake object will be lost after each move)
      • ex: { myFavoriteColor: "blue" }
    • getHeadCoords
      • type: function
      • returns object with coordinates of snake's head
      • ex: () => { row: 6, col: 6 }
    • getTailCoords
      • see "getHeadCoords"
    • getForwardTile
        • type: function returns object rdir: relative direction: "f" (forward), "l" (left), "r" (right) cdir: cardinal direction: "n", "e", "s", "w" row: number col: number letter: letter on map (see letters) ex: () => { rdir: "l", cdir: "e", row: 6, col: 12, letter: "*" }
    • getLeftTile
      see "getForwardTile"
    • getRightTile
      see "getForwardTile"
- map - utils `*` : apple `.` : empty `?` : unknown [a-zA-Z] : snake
